One: A Creative Habit on Display

To kick this whole year of creativity off, I've created the HTML and CSS structure of this site. You're looking at it right now! The structure doesn't make many stylistic choices, but it does lay the foundation for what's to come.

Its main page consists of a 52-item list with only one filled-in spot... so far. Each item of my 52-week journey will soon fill this space. Additionally, I have plans for this list to become dynamic and help track my progress. More on that later!

The foundation also includes a template for gallery pages such as this one. The primary goal of these gallery pages is to provide a flexible, branded expereience for housing my creative endeavours throghout 2020.

Here's some technical details: this whole site is hand-coded in HTML, CSS, and Javascript. The CSS has a framework in place called Milligram. Milligram is great because it provides some sane defaults for styling that I will build upon.

All of this code is stored on GitHub at https://github.com/alexmarc-us/Create-52.
